Detailed Description

La classe PcoCableCar implémente la classe abstraite CableCar, la classe PcoCableCar doit fournir les méthodes pour modéliser le comportement du télécabine ainsi que les méthodes qui permettent aux skieurs d'intéragir avec le télécabine.
